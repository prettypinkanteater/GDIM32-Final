### Group Devlog
We used sphere-casting to identify when the player is looking at, and nearby, an interactable object.
We wanted there to be items the player can interact with. Whether that be ingredients and utensils
to pick up or appliances to use with those items. We wanted the player to only be able to interact
with those items within a specified distance from themselves. Because a sphere-cast returns information 
abut the object it hit. We put this inside an if statement so that depending on the object it collided
with, we could get its tag and show a UI prompt using a switch statement. With the if statement, we can
use an else statement to hide the prompt if the sphere-cast doesn't detect anything.
